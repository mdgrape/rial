    Arith package provides floating-point arithmetics.

    Arith package provides floating-point arithmetics.

    For all the modules provided here, the following parameters can be customized by users: latency, floating point specification, and rounding mode.

    The modules take rial.arith.RealSpec of each argument, rial.arith.RoundSpec, and rial.util.PipelineStageConfig in their constructor. You can freely specify each parameters.
