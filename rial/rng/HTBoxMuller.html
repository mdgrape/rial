HTBoxMuller 

class HTBoxMuller extends Module

High-Throughput Box-Muller random number generator.

It uses polynomial approximations of math functions to calculate Box-Muller algorithm. It is fully pipelined without any stall but take much more resources compared to other Modules.

class HTBoxMuller(...) extends Module {
  val io = IO(new Bundle {
    val in = new Bundle {
      val ready = Output(Bool())
      val valid = Input(Bool())
      val x     = Input(UInt(cfg.rndW.W))
      val y     = Input(UInt(cfg.rndW.W))
    }
    val out = new Bundle {
      val ready = Input(Bool())
      val valid = Output(Bool())
      val z1    = Output(UInt(cfg.realSpec.W.W))
      val z2    = Output(UInt(cfg.realSpec.W.W))
    }
  })
}
